Мій електроний довідник HTML
Призначення Назва тегу Короткий опис Стандарт
Для карказу документа <html> Початок і закінчення HTML документа HTML5
<head> Вміщує опис веб-сторінки і являється контейнером для всіх заголовочних елементів html-документа HTML5
<body> Зберігання вмісту веб-сторінки, відображається у вікні браузера HTML5
Для роботи з текстом <p> Виявляє абзац тексту HTML5
<b> Жирний шрифт HTML5
<i> Курсивом(Italic) HTML5
<u> Горизонтальна лінія(підкреслення слова) HTML5
<ul> Неупорядкований список HTML5
<ol> Упорядкований список HTML5
<li> Елемент списка HTML5
<br> Відступи між абзацами HTML5
<h1-h6> Заголовки HTML5
<strong> Виявляє важливість тексту, виділяє жирним HTML5
<em> З наклоном, звертає увагу HTML5
Для гіперпосилань <a> Створює посилання HTML5
